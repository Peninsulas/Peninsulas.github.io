创建一个react项目模板

创建一个react项目模板

七月 23, 2019
  • newReact

一.准备工作

node.js

下载地址：https://nodejs.org/en/ （详见我的第一篇文章的第1节）

编辑器（vscode）

下载地址：https://code.visualstudio.com/

二.安装create-react-app

打开cmd，输入安装指令

三.创建React项目模板

在cmd中进入你想创建项目的路径，或者在文件夹中打开你想创建项目的文件夹，然后在上方输入cmd （详见我的第一篇文章的5.2章节）
输入创建指令

四.启动项目

1.可在编辑器打开项目并在终端中输入运行指令

终端在编辑器最上方的查看=>终端选项打开

2.可在cmd对应项目的根路径下输入运行指令

运行结果

这样一个React项目的模板就搭建完了，后续还需要加上Route（路由跳转），redux（数据管理），antd（ui样式模板），axios（请求）等等。我会在接下来的学习中边学边用边记录。
